# Batas Wilayah Layanan

`pinrang-service-boundaries.geojson` memuat subset 14 kelurahan pada Kecamatan
Paleteang dan Watang Sawitto dari data polygon yang diberikan untuk proyek.

Metadata pendamping:

- Dataset: Batas Wilayah Administrasi Desa/Kelurahan Indonesia
- Pemilik data: Badan Informasi Geospasial
- Edisi metadata: 28 September 2023
- Sistem koordinat: CRS84 / EPSG:4326
- Skala metadata: 1:10.000

Sesuai catatan metadata BIG, data dapat digunakan ketika batas definitif belum
tersedia, tetapi tidak boleh dianggap sebagai referensi hukum resmi.
